View Account Details

Introduction
The View Account Details screen displays the account details and security details linked to the account.
Navigation
  • Select Account Nickname hyperlink in the Account Summary screen.
  • The View Account Summary screen is displayed.
What you can do
The table below provides a brief description of the data that should be entered on the screen and the functions you can perform:.
Field
 Description
PERSONAL (CURRENCY) (ACCOUNT NUMBER)
GENERAL DETAILS
Account Nickname
 The short name assigned to the account. This can be modified using Update Account Preference under the Personal Profile menu option in the Non Financial Activities option of the user menu list.
Account Name
 The name of the account.
Account Status
 The status of the account, whether active, inactive and so on.
Account Type
 The type of operative account, whether savings, loan and so on
Account Category
 The scheme code or category under which the account has been opened.
Mode of Operation
 The mode of operation of the account.
Branch
 The branch where the account is held.
Account Open Date
The date on which the account was opened.
Currency
 The currency in which the account is held.
Primary Account
 Indicates if the account is a primary account. A primary account is an account which the customer normally operates.
For Example:
If the customer holds five different accounts, then the customer can designate one account as the primary account. When the customer logs on to the online banking system, the balances in your primary account are displayed by default. The customer can carry out transactions in this account. If the customer wants to carry out transactions or view balances in any other account, specify the account number.
